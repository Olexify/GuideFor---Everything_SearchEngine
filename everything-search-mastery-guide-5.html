01 · Foundation

Why Everything?

⚡
Instant Indexing
Reads NTFS MFT directly — indexes 1M+ files in under 1 second. Windows Search takes minutes.

6 MB RAM Only
Entire app uses ~6 MB RAM vs Windows Search consuming 200–800 MB. Stays resident in background.
🔎
Real-time Results
Results appear as you type — zero delay. Type 3 chars and see every match across all drives instantly.
🌐
Network + Remote
Built-in HTTP server, ETP protocol for cross-PC search. Serve your file index to any browser or phone.

10 · Content & Regex

Content Search & Regex

📄 content:
Searches inside file content (text files). Slower than name search — scans disk.
content:"api_key" *.env
🔡 utf8content:
Forces UTF-8 encoding for content search — use for Unicode text files, source code.
utf8content:"TODO" *.ts
⚙️ regex: in filenames
Apply full PCRE-style regex against the filename (not content). Instant — no disk scan.
regex:^(2025|2026).*\.pdf$
PDF files that start with 2025 or 2026
⚠️ Performance Note
content: reads every file — very slow on large drives. Always combine with file type or path filter: path:D:\Projects content:"TODO"

11 · Productivity

Filters, Bookmarks & Macros

🔖 Custom Filters
Create at Tools → Options → Filters. Assign a name and a search query. Filter appears in the dropdown left of the search box. Example: UE5 Configs → path:EpicGames *.ini
📌 Bookmarks
Save any search as a Bookmark via Bookmarks → Add Bookmark. Assign keyboard shortcut. Bookmarks persist across restarts and appear in the Bookmarks menu.
⌨️ Search Macros (Aliases)
Tools → Options → Search → Macros. Map a short keyword to a full query template.
ini → ext:ini
ue5 → path:E:\EpicGames
bigvid → video: size:huge
💾 Saved Searches (EFU)
Export any search as .efu file (File → Export). EFU = Everything File URL. Can be re-imported, shared, or used as a virtual folder in Everything.

13 · Network & Remote

Network, HTTP & ETP

🌐 HTTP Server
Enable at Tools → Options → HTTP Server. Set port (default: 80). Access from any browser on LAN: http://192.168.x.x:80
Works from phone, tablet, or other PCs. No app install needed. Supports search + file download links.
📡 ETP Protocol
Everything Transport Protocol — TCP-based. Lets another PC running Everything connect to your index. Enable at Options → ETP/FTP Server. From client PC: Tools → Connect to ETP Server.
🗂️ Network Drives (NAS)
Add UNC paths or mapped drives via Tools → Options → Indexes. Everything will index those shares too. Requires read permission on the share.
\\NAS01\Media → added as index
🔐 Security Note
HTTP server exposes all indexed file paths on your network. Set username/password in Options. Never expose port to the internet without auth.
